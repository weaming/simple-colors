# Simple Colors

Colorful output in terminal.

## Installation

```sh
pip install simple-colors
```

## Usage

```python
from simple_colors import *

print(green('hello'))
print(green('hello', 'bold'))
print(green('hello', 'underlined'))
```

The styles include:
* bold
* bright
* dim
* italic
* underlined
* blink
* reverse

## Inlucded colors

* black
* red
* green
